void spt_SparseTensorCollectZeros ( sptSparseTensor *  tsr)

Collect zeros from a sparse tensor

This function finds zero values from the sparse tensor, move them to the end of the array, then truncate the array. Orders are not kept, call sptSparseTensorSortIndex after this.
